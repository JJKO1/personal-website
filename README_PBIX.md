# Create PBIX (Power BI) using the provided theme and sample data

I cannot create a `.pbix` file directly in this environment because PBIX is generated by Power BI Desktop (a Windows/macOS GUI application). However, I prepared everything so you can create the PBIX in minutes.

Files added:

- [onepager.html](onepager.html) — English template for PDF export and preview.
- [powerbi_theme.json](powerbi_theme.json) — Power BI theme matching the orange + dark style.
- [sample_data.csv](sample_data.csv) — Example data to import into Power BI Desktop.

Quick steps (Power BI Desktop):

1. Open Power BI Desktop.
2. `Home` → `Get data` → `Text/CSV` → select `sample_data.csv`.
3. Load the data.
4. `View` → `Themes` → `Browse for themes` → select `powerbi_theme.json`.
5. Create visuals:
   - Card visuals: Revenue, ActiveUsers, ConversionRate.
   - Line chart: Date vs Revenue.
   - Donut/Pie: ChannelA/B/C distribution (use measure or columns accordingly).
   - Text box: Highlights and Recommended Actions (copy from `onepager.html`).
6. Arrange visuals to match the one-pager layout (one page sized canvas). Set page size to `16:9` or `Letter` depending on export needs.
7. Save report as `.pbix` (`File` → `Save`).
8. Export to PDF: `File` → `Export` → `Export to PDF` or use `Print` → `Microsoft Print to PDF`.
